# Быстрый деплой на Amvera

## Шаг 1: Подготовка репозитория

Убедитесь, что все изменения закоммичены:

```bash
git add .
git commit -m "Подготовка к деплою"
git push
```

## Шаг 2: Создание проекта на Amvera

1. Зайдите на https://amvera.ru/
2. Создайте новый проект
3. Выберите "Flask" или "Python"
4. Подключите ваш Git репозиторий

## Шаг 3: Настройка переменных окружения

В настройках проекта добавьте:

```
GIGACHAT_TOKEN=ваш_токен
MAX_BATCH_SIZE=30
REQUEST_TIMEOUT_SEC=60
```

(Остальные переменные опциональны)

## Шаг 4: Настройка запуска

Amvera должен автоматически определить `wsgi.py`. Если нет:
- **Точка входа:** `wsgi:app`
- Или используйте `Procfile` (уже создан)

## Шаг 5: Деплой

Amvera автоматически начнет деплой после подключения репозитория.

Проверьте:
- `/health` → должен вернуть `{"status":"ok"}`
- `/ui` → должна открыться главная страница

## Важно!

**Убедитесь, что `data/inputs_for_scoring.csv` включен в репозиторий:**
```bash
git add data/inputs_for_scoring.csv
git commit -m "Добавлен файл с данными"
git push
```

Если файл не включен, приложение не сможет загружать список сочинений.
